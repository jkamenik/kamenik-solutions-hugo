[Dev Container](https://containers.dev/) was a system born in [[VS Code]] that allowed the IDE to boot a container and use it as its development runtime. The benefits of this are huge: no longer does each developer need to set up a development machine, no longer does one developer's effort have to be manually replicated by all others, no longer is there a skew between dev, CI, and production. There simply is shared [[Declarative IaC|Infrastructure as Code]] (IaC) which is used to build all of those. And because of these benefits, Dev Container is very much in the adopt ring.

[DevOps](https://en.wikipedia.org/wiki/DevOps) is a development methodology that includes Operational concerns in the design and development process. It focuses on integrating [[Agile Software Development|Agile]] practices to ensure that value is delivered effectively. Often, [[Shift Left]] is used to describe these practices.
